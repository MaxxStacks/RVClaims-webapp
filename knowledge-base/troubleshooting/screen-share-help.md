# How to Share Your Screen with Support

DealerSuite360's screen sharing feature lets our support team see your screen in real time to help troubleshoot issues. No software downloads required — it runs entirely in your browser.

## When to Use Screen Share

Screen sharing is most useful when:
- You're having trouble with a specific step in a workflow and written instructions aren't enough
- An error is appearing on screen that's hard to describe
- Our team needs to walk you through a process in real time
- You want to confirm a claim has been set up correctly before submission

For simple questions, the Knowledge Base search or Live Chat in DS360 Assist is faster.

## How to Start a Screen Share Session

### Step 1: Open DS360 Assist

Click the **DS360 Assist** button in the bottom-right corner of any page. It looks like a help/chat bubble icon.

### Step 2: Go to Escalation Options

In the Assist panel, click **"Get More Help"** or **"Talk to a Person"** to open the escalation menu.

### Step 3: Select "Share My Screen"

Click the **Share My Screen** card. This generates a unique 6-character access code (format: XXX-XXX) that is valid for 10 minutes.

### Step 4: Share the Code with Support

Tell your support agent the access code — by phone, live chat, or email. The code is case-insensitive.

Your access code is displayed in large text in the Assist panel. A countdown timer shows how much time is left. If the code expires, you can generate a new one.

### Step 5: Select What to Share

When the support agent enters your code, your browser will ask you to choose what to share:

- **This Tab** (recommended) — shares only the DealerSuite360 browser tab
- **Entire Screen** — shares your full desktop
- **Other Window** — shares a specific application window

We recommend selecting **"This Tab"** to keep your other work private.

### Step 6: The Session Begins

Once you confirm what to share, the screen sharing session starts. A yellow banner will appear at the top of the screen showing the session is active.

You remain in full control at all times. You can end the session at any time by clicking **End Session** in the banner.

## Remote Control (Optional)

If you want our support team to demonstrate something directly, you can grant them temporary control of your mouse and keyboard. This is optional and must be explicitly approved by you:

1. The operator will request control — you'll see a dialog asking if you want to allow it
2. Click **Allow Control** to grant access, or **Deny** to keep view-only mode
3. While remote control is active, the banner turns orange/amber
4. You can revoke control at any time by clicking **Revoke Control** in the banner

**Safety note:** Our team can only see your DealerSuite360 session. They cannot access other applications, files on your computer, or browser tabs you haven't shared.

## Ending the Session

To end the screen share at any time:
- Click **End Session** in the yellow/green banner at the top of the page
- Or close the Assist panel (this ends the session automatically after a few seconds)

Once ended, the banner disappears and the support session is complete.

## Troubleshooting Screen Share Issues

**"My browser didn't ask me what to share"**
Your browser may have blocked the screen share prompt. Check for a blocked popup notification in the browser address bar and click "Allow."

**"The access code expired before my agent could connect"**
Generate a new code by clicking **Share My Screen** again. Codes expire after 10 minutes for security. Have your agent ready before generating the code.

**"The support agent can't see my screen"**
Make sure you completed Step 5 (selecting what to share in the browser dialog). The session won't start until you confirm what to share. If the problem persists, try refreshing the page and generating a new code.

**"I accidentally closed the Assist panel during a session"**
The banner at the top of the page stays visible even if the Assist panel is closed. You can still end the session using the **End Session** button in the banner.

## Browser Compatibility

Screen sharing requires a modern browser that supports the Web Share API:
- Google Chrome 72+ (recommended)
- Microsoft Edge 79+
- Firefox 66+
- Safari 13+ (limited support — Chrome is recommended for best experience)

Screen sharing is not supported on mobile browsers. Use a desktop or laptop computer for screen share sessions.
