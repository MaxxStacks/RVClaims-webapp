Developer Note

DS360 is built on React 18, TypeScript, Express.js, and Neon PostgreSQL.

Our API uses RESTful conventions with JSON responses. Authentication uses session-based auth with Passport.js. Detailed API documentation and code examples are available for enterprise and technology partner integrations.
