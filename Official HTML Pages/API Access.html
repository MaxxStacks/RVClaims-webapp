What You Can Build

The DS360 API gives you programmatic access to core platform data and operations.

DMS Integration

Sync unit data, customer records, and claim statuses between DS360 and your existing dealership management system. Eliminate double-entry and keep systems in sync.

Custom Reporting

Pull claim data, revenue metrics, and operational KPIs into your own business intelligence tools. Build dashboards that combine DS360 data with your other systems.

Workflow Automation

Trigger actions based on claim status changes, new unit additions, or customer events. Automate notifications, status updates, and cross-system workflows.

Partner Integrations

Insurance providers, parts suppliers, and service networks can integrate directly with DS360 to streamline transactions and data sharing with dealer partners.

Accounting Sync

Export invoice data, payment records, and claim financials to QuickBooks, Sage, Xero, or your preferred accounting platform via structured API responses.

Notification Webhooks

Receive real-time webhooks when claims change status, parts arrive, invoices are generated, or customers take action in their portal.

Getting Started

How to Get API Access

API access is available to enterprise dealers and approved technology partners. The process is straightforward and typically takes 2-3 business days.

Request Access

Submit an API access request via the contact form. Include your use case and integration requirements.

Review & Approval

Our team reviews your request and schedules a brief technical call to understand your integration needs.

Receive Credentials

Get your API key, client ID, and access to the full API documentation with sandbox environment.

Build & Launch

Build your integration with sandbox testing, then move to production when ready. Our team supports you through launch.

API Capabilities

  • Units — create, read, update unit records by VIN
  • Claims — submit, track, and manage claim lifecycle
  • FRC Lines — add, modify, and query failure repair codes
  • Customers — manage customer records and portal access
  • Invoices — generate, update, and export claim invoices
  • Parts — track parts orders linked to claims
  • Reports — pull aggregated operational data
  • Webhooks — subscribe to real-time event notifications

Enterprise-Grade Infrastructure

The DS360 API runs on the same infrastructure that powers the platform — hosted on Railway with Neon PostgreSQL, delivering 99.9% uptime with automatic scaling.

Authentication uses industry-standard API keys with optional OAuth 2.0 for partner integrations. All data is encrypted in transit and at rest.

Rate limits are generous for typical integrations and can be increased for high-volume enterprise use cases.

Frequently Asked Questions

Is API access included in all plans?
+
API access is available on Enterprise plans and for approved technology partners. Standard and Professional plans use the platform UI. Contact us to discuss API access for your specific needs.
Is there a sandbox environment?
+
Yes. Every API client receives sandbox credentials with test data. You can build and test your integration fully before connecting to production.
What are the rate limits?
+
Standard API access includes 1,000 requests per minute. Enterprise integrations can request higher limits based on their use case.
Do you support webhooks?
+
Yes. You can subscribe to webhooks for claim status changes, new units, customer actions, invoice generation, and more. Webhooks use standard HTTPS POST with JSON payloads.
